LastUpdated:
It can be any random dates in the format(dd/mm/yyyy) or (dd-mm-yyyy)
example : 22-10-2004   20/10/2004
List the tools updated after 22-10-2004
SELECT * FROM data WHERE LastUpdated > '22-10-2004'
List the tools updated before 22-10-2004
SELECT * FROM data WHERE LastUpdated < '22-10-2004'

Example : If a question contains a date then it should be used last in the query
list the tools updated before 04-07-2024 having MIT license in server side frmwrk
SELECT * FROM data WHERE License = 'MIT License' AND FunctionalArea = 'Server Side Framework' AND LastUpdated < '04-07-2024'
